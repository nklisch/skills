# Value vs. Effort Matrix: Know the Pitfalls and What to Do (Savio)

## Summary

A practitioner guide to the Value vs. Effort (Impact vs. Effort) 2x2 prioritization matrix, including the four quadrant labels, key pitfalls around subjective value estimation, and the conclusion that the matrix is a human-ceremony artifact rather than a machine-generated output.

## Key passages

**Four quadrants:**
- Quick Wins: High-value, low-effort — prioritize first.
- Big Bets: High-value, high-effort — tackle carefully.
- Fill-ins: Low-value, low-effort — use for roadmap padding.
- Time Sinks: Low-value, high-effort — deprioritize.

**Subjective value estimation (key pitfall):** The framework's biggest weakness is how practitioners assign value. The source notes that experts perform poorly "at estimating the value of features," often relying on gut feelings rather than data, using arbitrary numerical scales (1-5, 1-100) with minimal customer input.

**Planning fallacy in effort estimates:** Teams systematically underestimate time and resources needed (planning fallacy). Development teams should validate estimates before finalizing scores.

**Human ceremony requirement explicitly stated:** The matrix "requires manual scoring, validation discussions across teams, and regular re-evaluation as market conditions change. It's inherently a human-driven process requiring organizational alignment, not an automated calculation."

**Recommended fix:** Link value metrics to customer data — counting customers requesting a feature or aggregating their cumulative revenue — rather than relying on subjective assessments.

## Structural notes

- The 2x2 matrix produces a quadrant placement, not a continuous score. Items within the same quadrant have no relative ordering.
- Even when numeric scores are used internally (e.g., value 1-10, effort 1-10), the resulting ratio is comparable only across items evaluated in the same scoring session by the same group.
- The source explicitly labels the matrix as a "human-driven process" — it classifies the Value/Effort matrix on the ceremony end of the machine-readable/ceremony spectrum.
