# Programs

Megatopic research output lives here — one directory per program.

- **`<program-slug>/`** — produced by `/research-program` (`program.md` + `super-parent.md` + `campaigns/` + program evaluation)

Each campaign inside a program is itself a full `/deep-research` tree. Use this tier when a seed topic spans multiple distinct domains and a single `/deep-research` would be too narrow.
