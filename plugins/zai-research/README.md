# zai-research

Z.ai-powered research tools for Pi: web search, URL/PDF/JSON reading, and
GitHub repository deep reads through Z.ai's web-search-prime, web-reader, and
zread services.

## Install

```bash
pi install npm:@nklisch/pi-zai-research

# Local checkout/development install
pi install -l ./plugins/zai-research
```

The package's runtime extension requires the configured `zai` provider key in
Pi. See the `zai-research` skill for tool selection and source-verification
guidance.
