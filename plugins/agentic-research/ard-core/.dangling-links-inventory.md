## Re-scan command (F3 asserts clean after rewrite)

Run from `plugins/agentic-research/` — the COMPLETE scan (all `ard-core/**/*.md`,
not just the three docs the first F1 pass checked; widening to the full tree is
what caught `theory/README.md`):

```python
import re, os, glob
link_re = re.compile(r'\[[^\]]*\]\(([^)]+)\)')
for doc in glob.glob('ard-core/**/*.md', recursive=True):
    base = os.path.dirname(doc)
    for m in link_re.finditer(open(doc, encoding='utf-8').read()):
        tgt = m.group(1)
        if tgt.startswith('#') or '://' in tgt: continue
        p = tgt.split('#')[0]
        if p and not os.path.exists(os.path.normpath(os.path.join(base, p))):
            print(f"{doc} -> {tgt}")
```
