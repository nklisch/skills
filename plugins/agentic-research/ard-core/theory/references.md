# References

The consolidated bibliography for ARD's [defensibility trace](README.md) — every external source the [positions](positions/) rest on, de-duplicated into one numbered list. A position's `[handle]{N}` citation resolves to the correspondingly-numbered entry below.

This file is **bibliographic only**: author, title, venue, engagement scope, and substrate-confidence. It carries **no verbatim source reproduction** — load-bearing quotations live inside the positions that argue with them (bounded quotation for criticism and comment), not in a standalone extract layer. That is the deliberate IP posture: the *findings* that defend each commitment are ARD's own; the source attestations that reproduce external copyrighted passages stay in the originating deployment and are not republished here.

**On the `{N}`.** Each source has one **stable entry number** here, and every `[handle]{N}` citation across the positions uses it — so a handle carries the same `{N}` in every position, and `{N}` resolves to the entry of that number below. Numbers are **append-only**: a source added by extension takes the next integer and existing numbers never change, so live citations never break. (A handle's `{N}` is a whole-source pointer; where a position engages specific chapters or sections of a multi-piece source, the chapters are named in the prose and in the entry's engagement scope.)

**Substrate-confidence.** Entries engaged below source-direct depth (paywalled, bot-blocked, or read via secondary sources) are flagged inline. A load-bearing claim citing such a source carries a matching `{confidence: search-summary}` marker in the position body.

This list **grows by extension** as more commitments earn a ported rationale.
