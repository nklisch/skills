**v0.7.0 additions — grounding noted, positions not yet ported.** The engagement-economics arc rests on established external work, recorded here with positions deferred (curated, grows by extension). **`AQ.4`** (inadequate-attempt blocking) + the acquisition-mode-exhaustion fence ([§4.1](../SPEC.md#41-source-bound-citation-discipline)) draws on the information-seeking *stopping / abandonment* literature — good-vs-bad abandonment [li-good-abandonment-search]{47} and cognitive stopping rules [browne-cognitive-stopping-rules]{48} — read against the reliability distinction between transport-level (transient, retryable) and content-level (permanent) failure; it is the `AQ.3`-sibling extension of the acquisition discipline. (The abandonment literature grounds *that* premature stopping is a real failure; the specific transport-vs-content cut — session-gated 403 / anti-bot / JS-render as transport vs hard 404/410 as content-absence — is ARD's **operational** classification resting on standard transient-vs-permanent fault practice, not a tradition-grounded commitment, like the per-class IP profiles.) **`GR.9`** (metadata recall-sourcing) + the metadata-tier source-bound extension ([§4.1](../SPEC.md#41-source-bound-citation-discipline) / [§4.2](../SPEC.md#42-the-per-source-attestation-primitive)) rests on the measured finding that LLMs mis-fill bibliographic metadata of *real* sources — DOI accuracy ~16–20% on non-hallucinated references while titles run ~99% [chelli-hallucination-reference-accuracy]{45}, with ~24–43% of real citations carrying substantive metadata errors [walters-fabrication-citations-chatgpt]{46} — with the provenance-binding remedy framed by W3C PROV's entity/activity/agent model [w3c-prov-dm]{1}[w3c-prov-o]{2} (bind each metadata value to the fetch activity that produced it). The **model-diversity verification property** ([§7](../SPEC.md#7-the-verification-stack-the-rigor-control)) — the *partial* fence for the correlated-verifier limitation ([§11](../SPEC.md#11-known-limitations)) — rests on the evaluator-correlation literature: same-family judges share errors and self-preference [zheng-judging-llm-as-judge]{42}, intrinsic self-correction fails without an external signal [huang-llms-cannot-self-correct]{44}, and — the honesty caveat the fence carries — the most capable models correlate *even across providers* [kim-correlated-errors-llms]{41}, a special case of algorithmic-monoculture outcome homogenization [bommasani-picking-same-person-monoculture]{43}; cross-model review therefore *reduces*, never eliminates, shared blind-spots. The **`decision_relevance`** registration field + kickoff gate ([§9](../SPEC.md#9-the-registration-contract)) — the value-of-information yield gate — rests on decision-theoretic value of information [howard-information-value-theory]{49} and its expected-value-of-sample-information form [raiffa-schlaifer-applied-statistical-decision-theory]{51}, the marginal economics of search [stigler-economics-of-information]{50}, and sequential-analysis stopping rules [wald-sequential-analysis]{52}; the operative principle — *information has zero value if it cannot change the chosen action* — is the formal basis for the "what decision changes if this finds X?" test. The **`PR.3` class-complete sweep** ([§4.8](../SPEC.md#48-corrections-vs-reversals)) is a fence refinement grounded in the originating deployment's reconciliation-completeness evidence (a subset-flagged class whose unflagged siblings persist is the same non-propagation failure at the class level), not an external-tradition commitment. Substrate confidence on the new entries is **search-summary** (bibliographic existence verified; findings engaged via abstract / landing-page, not full source-direct reads) — itself an instance of the `GR.9` metadata discipline applied to ARD's own trace; de-coupled positions for these are not yet ported here.
